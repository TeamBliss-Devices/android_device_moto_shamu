# Device configuration for the Motorola Nexus 6 (shamu).

| Feature                 | Specification                     |
| :---------------------- | :-------------------------------- |
| CPU                     | Quad-core 2.7 GHz Krait 450       |
| Chipset                 | Qualcomm MSM8974AB Snapdragon 805 |
| GPU                     | Adreno 420                        |
| Memory                  | 3GB RAM                           |
| Shipped Android Version | 5.0                               |
